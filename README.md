# NER
To run ltp.py, you need to download ltp models and put it under ltp/ltp_data

```
......
├── bosonner.py
├── corenlp.py
├── foolnltk.py
├── ltp
│   └── ltp_data
│       ├── changelog
│       ├── cws.model
│       ├── md5.txt
│       ├── ner.model
│       ├── parser.model
│       ├── pos.model
│       ├── srl
│       │   ├── Chinese.xml
│       │   ├── prg.model
│       │   ├── srl.cfg
│       │   └── srl.model
│       └── version
├── ltp.py
....
```
